# 🏥 Clinic Management System

> Hệ thống quản lý phòng khám giúp bệnh nhân đặt lịch khám trực tuyến,
> bác sĩ quản lý lịch khám và admin quản lý toàn bộ hoạt động của phòng
> khám.

------------------------------------------------------------------------

# 📌 Overview

**Clinic Management System** được xây dựng nhằm giải quyết các vấn đề
phổ biến tại phòng khám truyền thống như:

-   ⏳ Bệnh nhân phải **xếp hàng chờ đợi lâu**
-   📅 Không biết **bác sĩ nào còn lịch trống**
-   📄 **Hồ sơ bệnh án dễ thất lạc**

Hệ thống giúp **số hóa toàn bộ quy trình khám bệnh** từ đặt lịch đến xem
kết quả xét nghiệm.

------------------------------------------------------------------------

# ✨ Key Features

-   📅 Online Appointment Booking
-   💳 Deposit Payment
-   📄 Electronic Prescription
-   🧪 Online Lab Results
-   ⭐ Doctor Rating System
-   📊 Admin Management & Reports

------------------------------------------------------------------------

# 👥 User Roles

  Role         Description
  ------------ --------------------------------------------------
  👤 Patient   Đặt lịch khám, xem đơn thuốc, kết quả xét nghiệm
  👨‍⚕️ Doctor    Khám bệnh, kê đơn thuốc, yêu cầu xét nghiệm
  🛠 Admin      Quản lý bác sĩ, chuyên khoa, thuốc, báo cáo

------------------------------------------------------------------------

# System Features
BỆNH NHÂN (6 tính năng): 
├── Đặt lịch khám online (chọn bác sĩ, ngày giờ) 
├── Thanh toán đặt cọc 
├── Xem lịch sử khám bệnh 
├── Xem đơn thuốc 
├── Xem kết quả xét nghiệm 
└── Đánh giá bác sĩ 
BÁC SĨ (6 tính năng): 
├── Xem lịch khám trong ngày 
├── Khám bệnh (ghi chẩn đoán) 
├── Kê đơn thuốc 
├── Yêu cầu xét nghiệm 
├── Xem lịch sử bệnh nhân 
└── Quản lý hồ sơ cá nhân 
ADMIN (6 tính năng): 
├── Quản lý bác sĩ (CRUD) 
├── Quản lý danh mục chuyên khoa 
├── Quản lý danh mục thuốc 
├── Cấu hình slot khám 
├── Báo cáo doanh thu 
└── Quản lý thông báo
